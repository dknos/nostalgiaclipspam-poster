# nostalgiaclipspam-poster

Personal automation tool for posting nostalgia / sports / educational short-form videos
to a single TikTok account ([@nostalgiaclipspam](https://www.tiktok.com/@nostalgiaclipspam)).

## What it does
Authenticates one TikTok account via OAuth, then uses the TikTok Content Posting API to
upload videos generated by an upstream pipeline (MLB Statcast clips, AI-generated
nostalgia interviews, etc.) on a scheduled cron.

## Scope
Single-account, single-creator. Does not access, store, or process any other user's data.
